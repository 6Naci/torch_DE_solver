# torch_DE_solver
Combines power of torch, numerical methods and math overall to conquer and solve ALL {O,P}DEs

There are several examples to provide a little insight to an operator form

* example_wave.py - Wave equation
* example_KdV.py - Korteweg- de Vries equation
* example_ODE_Legendre.py - Legendre polynomial generator equation
* example_Panleve_X.py - Panleve transcendents ODE
